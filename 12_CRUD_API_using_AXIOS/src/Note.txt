5. WHAT IS config IN AXIOS?
->Axios internally creates a config object.

->This config object contains EVERYTHING about that request:
URL
Method (GET, POST, PUT...)
Headers
Timeout
Body / Data
Params
Base URL
Authentication
Interceptor flags
Transform functions

->It is like a package of full request details.

->EXAMPLE
    If you run:  api.get("/posts");
    Axios will create this config:
    {
        baseURL: "https://jsonplaceholder.typicode.com",
        url: "/posts",
        method: "get",
        headers: {
            Accept: "application/json",
            "Content-Type": "application/json",
            Authorization: "Bearer YOUR_TOKEN"   // added by interceptor
        },
        timeout: 5000,
        data: undefined,
        params: undefined,
    }
